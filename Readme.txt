Данный проект реализует функциональность приложения к T-FLEX CAD
В этом приложении создаются объекты с новым поведением (звёзды)
Приложение после компиляции может быть добавлено и запущено из
диалога "Приложения..." T-FLEX CAD.

При сборке проекта на конкретном рабочем месте необходимо обратить внимание 
на расположение файла, на который ссылается проект (tflexapi.dll). 
Если он не находится, то нужно удалить данную ссылку и добавить ссылку на существующий
файл, например C:\Program Files\T-FLEX\T-FLEX CAD 12\Program\TFlexAPI.dll.
Обязательно свойство CopyLocal после этого нужно установить в значение false.

Привет!
